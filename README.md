🚦 New York Road Accidents Analysis | Tableau Dashboard

📌 Project Overview
This project analyzes road accident data in New York to identify patterns, accident trends, and key risk factors.  
The interactive dashboard helps understand accident distribution by location, time, and contributing factors.

The goal of this project is to demonstrate data visualization and analytical skills using Tableau.

📊 Dashboard Insights
Key insights generated from the dashboard:

- Accident distribution by borough/location
- Accident trends over time
- Impact of weather and road conditions
- Vehicle type involved in accidents
- Peak accident hours and days
